# Weather App Project

A dynamic weather application built with HTML, CSS, and JavaScript that fetches real-time weather data using a weather API.

## 🌤️ Live Demo

[View the Weather App](https://korirleonard.github.io/weather-app-project-/)

## Features

- Real-time weather data
- Search weather by city
- Display current temperature, humidity, and weather conditions
- Responsive design for mobile and desktop
- Clean and intuitive user interface
- API integration with weather service

## Technologies Used

- **HTML5** — Structure
- **CSS3** — Styling and responsive design
- **JavaScript** — API calls and dynamic updates
- **Weather API** — Real-time weather data (OpenWeatherMap or similar)

## Getting Started

### View the Project

1. Visit the [live demo](https://korirleonard.github.io/weather-app-project-/)
2. Or clone the repository:

```bash
git clone https://github.com/KorirLeonard/weather-app-project-.git
cd weather-app-project-
```

3. Open `index.html` in your browser
4. Enter a city name to get weather information

## API Integration

This project uses a weather API to fetch real-time data:
- Search by city name
- Get current temperature
- Display weather conditions
- Show humidity and other details

## How It Works

1. User enters a city name in the search box
2. JavaScript fetches weather data from the API
3. App displays current weather conditions
4. Information updates dynamically without page reload
